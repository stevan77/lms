LMS - Sistem za upravljanje ucenjem

Kompletna platforma za skole sa vise uloga, kursevima, zadacima i pracenjem napretka ucenika.

Uloge korisnika

A

Admin

Upravlja razredima, kursevima, nastavnicima i ucenicima unutar svoje skole.

N

Nastavnik

Kreira sadrzaj lekcija, prati napredak ucenika, ocenjuje zadatke, salje poruke.

U

Ucenik

Ucenje kroz lekcije, izvrsavanje zadataka, pracenje sopstvenog napretka.

Kljucne mogucnosti

Rich Text Editor

TipTap editor sa formatiranjem, tabelama, slikama, blokovima koda i jos mnogo toga.

Python u browseru

CodeMirror editor + Pyodide izvrsavanje. Podrska za input(), Pygame emulaciju i matplotlib.

LaTeX matematika

KaTeX renderovanje formula. Podrska za sisteme jednacina, razlomke, matrice.

Jupyter Notebook

Markdown + code celije sa izvrsavanjem, matplotlib graficima koji se renderuju kao PNG.

Zadaci i ocenjivanje

Lekcije mogu biti zadaci. Ucenik predaje, nastavnik ocenjuje (1-5), vraca na doradu.

Poruke i notifikacije

Direktne poruke, broadcast po razredu, zvonce sa notifikacijama u realnom vremenu.

Admin

Administrator skole. Upravlja razredima, kursevima, nastavnicima i ucenicima.

Dashboard

Statistika: broj razreda, kurseva, nastavnika i ucenika u skoli za izabranu skolsku godinu.

Razredi (Grades)

PoljeOpis
NazivNpr. "6-1", "7a", "Clasa a 5-a"
NivoBroj razreda (1-8), koristi se za promociju
JezikSrpski, Rumunski ili Engleski - odredjuje jezik interfejsa za ucenike
Token za upisAutomatski generisan. Deljiv link za samostalni upis ucenika

Kursevi (Courses)

Kreiranje kurseva sa nazivom i opisom. Kurs se dodeljuje:

  • Razredu - Svi ucenici razreda dobijaju pristup kursu. Bira se i nastavnik.
  • Pojedinacnom uceniku - Individualna dodela sa filterom po razredu. Bira se nastavnik.
Na stranici za uredjivanje kursa, sekcija "Dodeljeni razredi" prikazuje razrede (ljubicasti badge), a "Dodeljeni ucenici" prikazuje individualne dodele (zeleni badge).

Nastavnici i ucenici

CRUD za nastavnike i ucenike. Ucenici se dodeljuju razredima. Nastavnici se dodeljuju kursevima preko Course-Grade veze.

Selektor skolske godine

U headeru se nalazi dropdown za izbor skolske godine. Svi podaci (razredi, kursevi, ucenici) se filtriraju po izabranoj godini. Zakljucane godine su read-only.

Nastavnik

Kreira sadrzaj, prati napredak, ocenjuje zadatke, komunicira sa ucenicima.

Dashboard

Prikaz svih kurseva sa brojem lekcija i ucenika. Za svaki kurs prikazani su dodeljeni razredi (ljubicasto) i individualni ucenici (zeleno). Linkovi za napredak po razredu/uceniku.

Upravljanje lekcijama

Hijerarhijska struktura kursa:

Kurs
Kapitola (Chapter)
Podkapitola (Subchapter)
Lekcija (Lesson)
  • Kapitole i podkapitole se kreiraju inline (naziv + Enter)
  • Lekcije se kreiraju na posebnoj stranici sa Rich Text Editorom
  • Drag & drop za promenu redosleda kapitola, podkapitola i lekcija
  • Preview dugme za prikaz lekcije iz perspektive ucenika
  • Lekcija moze biti oznacena kao Zadatak

Pracenje napretka

Stranica sa karticama za svakog ucenika. Svaka kartica prikazuje mini tackice za svaku lekciju:

  • Siva = Nije zapoceto
  • Zuta = U toku
  • Zelena = Zavrseno

Zadaci (Submissions)

Dva nacina pristupa zadacima:

  • Per lekcija - Iz lekcije klik na "Submissions" dugme za konkretni razred
  • Globalni pregled - "Zadaci" u sidebaru (sve predaje svih ucenika)

Globalni pregled ima filtere po razredu i imenu ucenika. Podrazumevano prikazuje samo neocenjene. Dugme "Prikazi ocenjene" toggleuje prikaz.

Za svaku predaju nastavnik moze:

  • Oceniti (1-5) sa feedback-om - lekcija se automatski oznacava kao zavrsena
  • Vratiti na doradu sa komentarom - ucenik ponovo predaje

Poruke

Direktne poruke uceniku ili broadcast celom razredu. Inbox sa filterima po uceniku/razredu.

Ucenik

Ucenje, predaja zadataka, pracenje napretka.

Moji kursevi

Prikaz svih kurseva u kartici sa gradient headerom. Progress bar i broj zavrsenih lekcija za svaki kurs. Kursevi dolaze iz dva izvora:

  • Preko razreda (svi kursevi dodeljeni razredu)
  • Individualna dodela (badge "Individualno")

Ucenje lekcije

Stranica kursa prikazuje kapitole/podkapitole/lekcije sa collapsible strukturom. Aktivna lekcija se automatski otvara i skroluje do nje pri ucitavanju.

Tok ucenja:

Nije zapoceto
→
U toku
→
Zavrseno
  • Otvaranje lekcije automatski je oznacava kao "U toku"
  • Dugme "Oznaci kao zavrseno" - automatski otvara sledecu lekciju sa scroll-om
  • "Vrati u toku" - moze ponisti zavrsetak (undo)

Zadaci

Zadaci se predaju inline u lekciji (textarea + dugme). Statusi predaje:

StatusOpis
PredatoCeka pregled nastavnika
OcenjenoNastavnik ocenio (1-5), lekcija automatski zavrsena
PregledanoNastavnik pregledao bez ocene
VracenoNastavnik vratio na doradu - ucenik ponovo predaje

Moji zadaci

"Moji zadaci" u sidebaru prikazuje sve predaje sa filterima po kursu i statusu. Statistika: ukupno, ceka pregled, ocenjeno, vraceno. Klik otvara detalje sa ocenom i feedback-om.

Pitaj nastavnika

Na SVAKOJ lekciji (ne samo zadacima) postoji dugme "Pitaj nastavnika". Inline forma sa textarea - poruka se salje bez napustanja stranice. Potvrdna poruka "Poruka poslata!" na 5 sekundi.

Renderovanje za ucenika

Komponenta LessonContent.vue parsira HTML sadrzaj i automatski prepoznaje:

  • data-code-exercise atribute - renderuje CodeExerciseRunner
  • data-notebook atribute - renderuje NotebookRunner
  • \[...\] delimitere - renderuje KaTeX matematiku
  • Obican HTML - renderuje standardno

Python vezbe

Interaktivne Python vezbe sa izvrsavanjem u browseru.

Kako radi

Nastavnik kreira vezbu u editoru
→
Starter code + instrukcije
→
Ucenik menja kod
→
Pyodide izvrsava u browseru

Mogucnosti

  • CodeMirror editor - Syntax highlighting, line numbers, Python autocomplete
  • Pyodide (WASM) - Kompletan Python interpreter u browseru, bez servera
  • input() podrska - Prompt dialog za korisnicki unos
  • Prijateljske greske - Python greske prevedene na srpski za decu
  • Reset dugme - Vracanje na originalni starter kod

Pygame emulator

Pygame shim (public/pygame-shim.py) koji mapira Pygame pozive na HTML Canvas:

  • pygame.draw.line() - Linije
  • pygame.draw.circle() - Krugovi
  • pygame.draw.rect() - Pravougaonici
  • pygame.draw.ellipse() - Elipse
  • pygame.draw.polygon() - Poligoni
  • pygame.draw.arc() - Lukovi
  • Tekst renderovanje na Canvas-u

Matematika (LaTeX)

KaTeX renderovanje matematickih formula u lekcijama.

Nacini koriscenja

NacinSintaksaRezultat
Inline formula\[x^2 + y^2 = z^2\]Formula u tekstu
Math blok u editoruMathExtension nodeIzolovani math blok
Mesoviti tekst+math\[...\] delimiteri u HTML-uTekst sa formulama

Primeri sadrzaja

  • Sistemi linearnih jednacina sa \begin{cases}
  • Razlomci, koreni, eksponenti
  • Matrice i determinante
  • Step-by-step resenja (5 koraka: nepoznate, jednacine, sistem, resavanje, provera)
U HTML atributima, \begin{cases} zahteva \\\\ za prelom reda (dvostruko eskejpovanje).

Jupyter Notebook

Interaktivni notebook sa markdown i code celijama.

Mogucnosti

  • Markdown celije - Tekst sa formatiranjem
  • Code celije - Python kod sa izvrsavanjem (Pyodide)
  • matplotlib - Grafici se renderuju kao PNG slike
  • Run All - Izvrsavanje svih celija odjednom
  • Per-cell execution - Izvrsavanje pojedinacne celije
  • Output se prikazuje ispod svake celije

Poruke i notifikacije

Komunikacija izmedju nastavnika i ucenika.

Poruke

  • Direktna poruka - Nastavnik ↔ ucenik, obostrana komunikacija
  • Broadcast - Nastavnik salje celom razredu, sa tabelom ko je procitao
  • Inbox - Filtere po uceniku (za nastavnika) ili kursu
  • Brisanje - Soft delete po korisniku (drugi korisnik i dalje vidi poruku)
  • Badge - Broj neprocitanih poruka u sidebaru

Skolske godine

Upravljanje skolskim godinama sa promocijom ucenika.

Struktura

PoljeOpis
nameNpr. "2025/2026"
start_date / end_dateDatum pocetka i kraja
is_activeSamo jedna godina moze biti aktivna
is_lockedZakljucane godine su read-only

Filtriranje po godini

Selektor u headeru (vidljiv adminu i nastavniku). Svi podaci se filtriraju:

  • Razredi imaju school_year_id
  • Kursevi se filtriraju preko razreda
  • Individualne dodele imaju direktan school_year_id
  • Admin dashboard, teacher dashboard, student dashboard - sve je filtrirano

Zatvaranje godine (wizard)

  1. Preview - Pregled svih promena pre zatvaranja
  2. Promocija - Level ucenika + 1 (npr. 5. razred → 6. razred)
  3. Maturanti - Ucenici 8. razreda dobijaju status "graduated"
  4. Zakljucivanje - Stara godina locked, nova kreirana i aktivirana

Podrzani jezici

Ukupno 363 kljuca za prevod po jeziku.

Kako radi

  • Vue strana - useTranslations composable sa t('key') funkcijom
  • Laravel strana - SetLocale middleware cita korisnikov jezik
  • Promena jezika - Dropdown u sidebaru, cuva se u bazi (users.locale)
  • Auto-detekcija - Razred ima locale polje. Upis ucenika automatski postavlja jezik.
Kursevi na razlicitim jezicima: "Osnove programiranja u Python-u" (SR) i "Bazele programarii in Python" (RO) - isti sadrzaj kompletno preveden ukljucujuci print stringove, komentare i imena promenljivih u kodu.

Tok upisa

Admin kreira razred
→
Generise se token
→
Deli link ucenicima
→
Ucenik se registruje

Svaki razred ima jedinstven enrollment_token. URL format: /enroll/{token}

Pri registraciji, sistem automatski:

  • Kreira korisnicki nalog (ucenik)
  • Dodeljuje uceniku razred
  • Postavlja skolu
  • Postavlja jezik prema podesavanju razreda
